Projeto TeenProtect

Antes de abrir o arquivo, verifique se você tem o node.js instalado no pc (caso não tenha instalado, baixar e instalar o node.js na página oficial deles)
Execute o comando 'node server.js' no terminal do vscode (Caso no terminal exibir uma mensagem 'servidor rodando' isso significa que o servidor do site está rodando).
Depois vá no navegador e escreva localhost:3000 para entrar e abrir o arquivo e visualizar o site.
